# Postman Customer Success Engineering

We build open-source tooling and automation for API lifecycle management, helping enterprise teams onboard, govern, and scale their API programs with Postman.

## Open Source Tools

### GitHub Actions

| Action | Description |
|--------|-------------|
| [postman-api-onboarding-action](https://github.com/postman-cs/postman-api-onboarding-action) | End-to-end Postman onboarding orchestration |
| [postman-insights-onboarding-action](https://github.com/postman-cs/postman-insights-onboarding-action) | Onboard discovered services to API Catalog workspaces |
| [postman-aws-spec-discovery-action](https://github.com/postman-cs/postman-aws-spec-discovery-action) | Discover AWS API Gateway APIs and export OpenAPI 3.0 specs |
| [postman-bootstrap-action](https://github.com/postman-cs/postman-bootstrap-action) | Workspace, spec, and collection bootstrap |
| [postman-repo-sync-action](https://github.com/postman-cs/postman-repo-sync-action) | Repo sync, environments, and Bifrost workspace linking |
| [postman-api-gateway-sync](https://github.com/postman-cs/postman-api-gateway-sync) | Sync OpenAPI specs from AWS API Gateway to Postman Spec Hub |

### CLI & Editor Tools

| Tool | Description |
|------|-------------|
| [postman-cli-usage-reports](https://github.com/postman-cs/postman-cli-usage-reports) | Fetch rich, detailed Postman reports via CLI |
| [postman-clx](https://github.com/postman-cs/postman-clx) | Postman CLI with extended reporting |
| [postman-vscode-lint](https://github.com/postman-cs/postman-vscode-lint) | Real-time API governance validation for VS Code |

### Governance & Automation

| Tool | Description |
|------|-------------|
| [governance-automation](https://github.com/postman-cs/governance-automation) | API governance automation with dual-mode support for specs and APIs |
| [bidirectional-spec-to-postman](https://github.com/postman-cs/bidirectional-spec-to-postman) | OpenAPI-to-Postman bidirectional sync with contract tests |
| [SAMLite](https://github.com/postman-cs/SAMLite) | Service account authentication for Postman via local SAML IdP |
